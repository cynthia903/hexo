| 1992 | [SQL-92](https://en.wikipedia.org/wiki/SQL-92)     | SQL2, FIPS 127-2                                             | 引入事务隔离级别等特性 |
| 1999 | [SQL:1999](https://en.wikipedia.org/wiki/SQL:1999) | SQL3                                                         |                        |
| 2003 | [SQL:2003](https://en.wikipedia.org/wiki/SQL:2003) |                                                              |                        |
